## インストール

    $ https://github.com/k-ishiwata/Workdock.git
    $ cd Workdock
    $ composer install
    $ cp .env.example .env
    $ php artisan key:generate
    $ npm install
    $ npm prod

## データベース設定

    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=homestead
    DB_USERNAME=homestead
    DB_PASSWORD=secret
